Proposito del Sitio:

El propósito de este sitio web es aplicar y demostrar la aplicación de los fundamentos de HTML, ya que con ellas podemos estructurar y desplegar una página web. Además podremos mejorar el contenido de una página web, ya que de esta manera se puede aumentar la interacción que un humano tiene con el ordenador, por lo tanto se vuelve llamativa la experiencia de la página y más sencilla la forma de inferir la información que se da a conocer.

Introducción HTML:

"Crear una pagina web es más sencillo de lo que parece. En realidad, cualquiera puede crear o editar una web. No es necesario tener ninguna habilidad especial, ni haber estudiado un máster, para desarrollar paginas web; todo lo que se necesita es una gran voluntad de aprender, la motivación que realmente puedes conseguir, y la paciencia sufiencte para ir avanzando en el proceso de publicación de una web en internet."
